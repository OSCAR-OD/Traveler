Traveler, built with ASP.NET Web API and a layered architecture (Controllers, BLL, DTOs) to manage travel-related services.
Key Features:
Retrieve all available tour packages.
View the complete customer list.
Get individual customer details by ID.
Traveler login authentication system.
Structured DTO models for Customers, Bookings, and Payments.

Technologies Used:
ASP.NET Web API
C#
Entity Framework
Layered Architecture (Controller → Service → DTO)
