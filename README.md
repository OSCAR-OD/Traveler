Traveler, built with ASP.NET Web API, Microsoft SQL Server, a layered architecture (Controllers, BLL, DTOs) to manage travel-related services.

Key Features:
1. Retrieve all available tour packages.
2. View the complete customer list.
3. Get individual customer details by ID.
4. Traveler login authentication system.
5. Structured DTO models for Customers, Bookings, and Payments.

Technologies Used:
1. ASP.NET Web API
2. C#
3. Entity Framework
4. Layered Architecture (Controller → Service → DTO)
